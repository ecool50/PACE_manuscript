path above, not a second copy. Upload the real file.

> **Re-hash before uploading.** The md5 above describes the file as it currently sits on
> disk. If the SPE is re-saved for any reason the hash changes, and `fetch_data.R` will
> reject the download. Regenerate with `md5 -q <path>` (macOS) / `md5sum <path>` (Linux)
> and use whatever the uploaded file hashes to.

Too large for git: GitHub hard-rejects any file over 100 MB. Git LFS's free tier (1 GB
bandwidth/month) would cut off after ~7 clones. Zenodo allows 50 GB per record and mints
a DOI, which is what `methods.tex` needs anyway.

> Janesick et al. (2023), *Nature Communications*. The value added here is the attached
> scClassify cell-type annotation and per-cell quality-control metrics.
